Clipboard Utility is a clipboard utility tool designed to help format text for translating academic papers. It allows you to remove line breaks from your copied text with a single click.

## Features

* Displays the character count near the mouse cursor when the clipboard text changes.
* Allows you to manipulate the clipboard text from the task tray icon.

## Installation

### 1\. Installer (Recommended)

You can download the latest installer from here:
[![Latest Release](https://img.shields.io/github/v/release/GK-302/Clipboard-Utility)](https://github.com/GK-302/Clipboard-Utility/releases/latest)

The release page provides the following three types depending on your environment:

  * **Standard Version:** `ClipboardUtility-Installer-vX.X.X.zip`
      * The standard installer. Administrator privileges are recommended.
  * **No-Admin Version:** `ClipboardUtility-Installer-NoAdmin-vX.X.X.zip`
      * Can be installed without administrator privileges, though the installation location is restricted.
  * **Exe Version (Portable):** `ClipboardUtility-Standalone-vX.X.X.zip`
      * No installation required; runs as a standalone executable.

-----

### 2\. Other Methods

  * **Manual Startup Registration**:
      * If you use the `Exe Version` and want the tool to launch automatically on PC startup, please manually add the executable file to your OS startup folder.
  * **Building from Source**:
      * You can also build and use the application from the source code.

## Usage

1. Launch the application.
2. When you copy text, the character count will appear near your mouse cursor.
3. Click the task tray icon to execute registered actions.
4. Right-click the icon to open the menu.
